# Descrierea Proiectului

Acest proiect este o aplicatie simplã dar utilã pentru gestionarea unui numãr de saci si cutii, precum si
 a continutului acestora, în special atunci când te muti. Aplicatia este usor de utilizat, permitând 
introducerea de cifre si text conform instructiunilor.

## Functionalitãti

- **Memorare**: Adaugã saci si cutii în sistem.
- **?tergere**: Eliminã saci, cutii sau continutul acestora.
- **Afi?are**: Vizualizeazã lista de saci si cutii, împreunã cu continutul acestora.
- **Resetare**: Reseteazã complet sistemul, eliminând toate datele.

## Utilizare

Introduceti datele conform instructiunilor afisate pentru a adãuga, sterge, vizualiza sau reseta informatiile.
